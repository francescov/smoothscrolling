How to make smooth scrolling.

To enable smooth scrolling, all you need to do is add the "rel" attribute with value "relativeanchor". The link destination should be a page anchor.

The page will gracefully degrade to default behavior and jump, but with jQuery available, you can progressively enhance and enable the animation jump.
